## About

Cocktail Page is a modern recipe browsing application where users can explore cocktails, view details, and search drinks instantly.
Built using ReactJS with smooth GSAP animations, it delivers a clean, fast, and enjoyable recipe-exploration experience.

## Getting Started

### Installation

1. Clone the repo:

   1. git clone https://github.com/Nasimul650/coktail-page

   2. cd coktail-page
   3. npm install
   4. npm run dev
   5. Go to "http://localhost:5173" in your browser
   6. npm run dev

---

## Usage

1. Browse all cocktails on the home page.

2. Search drinks by name using the search bar.

3. Click a drink to view full recipe details.

4. Enjoy smooth animations powered by GSAP.
